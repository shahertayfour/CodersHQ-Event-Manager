  CHQ SPACE MANAGEMENT PLATFORM - HOW TO RUN
═══════════════════════════════════════════════════════════════

🎯 EASIEST METHOD (Windows):

   1. Make sure PostgreSQL is installed and running

   2. First time only - Right-click "first-time-setup.ps1"
      → Select "Run with PowerShell"

   3. To start the app - Right-click "start-dev.ps1"
      → Select "Run with PowerShell"

   4. Open browser to: http://localhost:3000

   5. Login as admin:
      Email:    admin@codershq.ae
      Password: Admin@123456

═══════════════════════════════════════════════════════════════

📋 MANUAL METHOD:

   STEP 1 - Setup PostgreSQL Database:

   • Open pgAdmin or psql
   • Create database: chq_space_management
   • Or use existing PostgreSQL database

   ─────────────────────────────────────────────────────────────

   STEP 2 - Setup Backend (First Time Only):

   • Open Command Prompt
   • Run these commands:

     cd "C:\Users\User\Desktop\claude\CodersHQ event Registration\backend"
     npm install
     copy .env.example .env

     (Edit .env file with your database settings)

     npm run prisma:generate
     npm run prisma:migrate
     npm run prisma:seed

   ─────────────────────────────────────────────────────────────

   STEP 3 - Setup Frontend (First Time Only):

   • Open ANOTHER Command Prompt
   • Run these commands:

     cd "C:\Users\User\Desktop\claude\CodersHQ event Registration\frontend"
     npm install

   ─────────────────────────────────────────────────────────────

   STEP 4 - Start Backend (Every Time):

   • In first Command Prompt:

     cd "C:\Users\User\Desktop\claude\CodersHQ event Registration\backend"
     npm run start:dev

   • Wait until you see: "Application is running on: http://localhost:4000/api"

   ─────────────────────────────────────────────────────────────

   STEP 5 - Start Frontend (Every Time):

   • In second Command Prompt:

     cd "C:\Users\User\Desktop\claude\CodersHQ event Registration\frontend"
     npm run dev

   • Wait until you see: "Ready on http://localhost:3000"

   ─────────────────────────────────────────────────────────────

   STEP 6 - Access Application:

   • Open browser to: http://localhost:3000

   • Admin Login:
     Email:    admin@codershq.ae
     Password: Admin@123456

═══════════════════════════════════════════════════════════════

🛑 TO STOP:

   • Press Ctrl+C in both Command Prompt windows
   • Or close the windows

═══════════════════════════════════════════════════════════════

⚠️ TROUBLESHOOTING:

   Problem: Backend won't start
   Solution:
   • Make sure PostgreSQL is running
   • Check backend\.env file exists and has correct DATABASE_URL
   • Port 4000 must be free

   ───────────────────────────────────────────────────────────

   Problem: Frontend won't start
   Solution:
   • Make sure backend is running first
   • Port 3000 must be free

   ───────────────────────────────────────────────────────────

   Problem: Database errors
   Solution:
   • cd backend
   • npx prisma migrate reset
   • npm run prisma:seed
